## 3. 目录职责说明表

| 目录 | 职责 | 对应计划 |

| api/ | REST API 与鉴权 | EP-03 |
| executor/ | 执行引擎与状态机 | EP-04 |
| streaming/ | WebSocket 与 HIL | EP-05 |
| report/ | 报告生成与导出 | EP-06 |
| security/ | 安全与多租户 | EP-08 |
| observability/ | 日志与指标 | EP-09 |
