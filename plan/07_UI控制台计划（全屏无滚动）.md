# Plan 07：UI 控制台计划（全屏无滚动）

## 目标

- 落地 SaaS 控制台核心页面：任务、实例 Live Console、HIL、报告
- 强制全屏无滚动条：所有信息以分页/折叠/抽屉呈现

## 依赖关系

本计划依赖以下端点计划的实现：

| 依赖项 | 说明 |
|--------|------|
| EP-03 (控制面 API) | 提供任务/实例/报告的 CRUD 接口 |
| EP-05 (WebSocket 推流) | 提供实例截图流、动作事件实时推送 |
| EP-06 (报告数据) | 提供测试报告结构化数据与回放能力 |

## 范围

### 页面（P0）

- 任务列表 + 创建向导（可延后为 P1）
- 实例 Live Console（截图+标注层+动作卡+日志+时间线）
- HIL 工单抽屉（approve/reject/modify，JSON 截断+复制）
- 报告回放（摘要/缺陷列表/时间线回放/JSON 导出）

### 全屏无滚动条规则（P0）

- 浏览器级：`html, body { height:100%; overflow:hidden; }`
- 容器级：任何面板禁止 `overflow:auto/scroll`
- 列表/日志/时间线：固定行数 + 上一页/下一页
- 长文本/JSON：截断预览 + 复制全文（不出现滚动条）

## Harness（先做）

### H1：No-Scroll Harness（P0）

运行时断言与 CSS 静态检查，确保全屏无滚动条。

**测试用例：**

- `should_have_no_browser_scrollbar` - 浏览器级无滚动条
  - 断言 `document.documentElement.scrollHeight === clientHeight`
  - 断言 `body.scrollHeight === clientHeight`
- `should_have_no_container_scrollbar` - 容器级无滚动条
  - 断言所有 `.panel`, `.sidebar`, `.content` 容器无 `overflow:auto/scroll`
  - 静态 CSS 检查禁止规则
- `should_fit_viewport_at_any_resolution` - 任意分辨率适配
  - 测试 1920x1080、1366x768、2560x1440 等常见分辨率
  - 断言布局无溢出
- `should_hide_overflow_content` - 溢出内容隐藏
  - 断言超长内容被截断而非滚动
  - 验证截断提示（如 "..." 或 "更多"）

### H2：UI Paging Harness（P0）

边界与交互测试，确保分页替代滚动。

**测试用例：**

- `should_paginate_list_items` - 列表项分页
  - 给定 N 条数据（N > pageSize）
  - 断言仅显示 pageSize 条
  - 断言分页器显示总页数
- `should_navigate_between_pages` - 页间导航
  - 点击下一页加载下一批数据
  - 点击上一页返回前一批数据
  - 首页禁用上一页，末页禁用下一页
- `should_display_page_info` - 分页信息显示
  - 显示 "第 X/Y 页"
  - 显示 "共 N 条"
  - 可选：每页条数切换
- `should_handle_empty_list` - 空列表处理
  - 空数据时显示空态占位
  - 分页器隐藏或禁用

### H3：Overlay Harness（P0）

标注层绘制测试，确保截图标注正确。

**测试用例：**

- `should_render_bbox_on_screenshot` - 截图上绘制边界框
  - 给定 bbox fixture `{x, y, width, height}`
  - 断言 canvas/SVG 正确绘制矩形
  - 断言边界框位置与尺寸正确
- `should_handle_multiple_overlays` - 多标注处理
  - 给定多个 bbox/point fixtures
  - 断言所有标注同时渲染
  - 断言标注层级/颜色区分
- `should_clear_overlays_on_step_change` - 步骤切换清空
  - 切换时间线步骤时
  - 断言旧标注清空
  - 断言新标注正确渲染

## 全屏无滚动条 CSS 规范

```css
/* 全局无滚动条规则 */
html, body {
  height: 100%;
  overflow: hidden;
  margin: 0;
  padding: 0;
}

/* 容器级无滚动条规则 */
.panel, .sidebar, .content {
  height: 100%;
  overflow: hidden;
}

/* 列表/日志分页替代滚动 */
.paginated-list {
  display: flex;
  flex-direction: column;
  height: 100%;
}

.list-content {
  flex: 1;
  overflow: hidden;  /* 禁止滚动 */
}
```
